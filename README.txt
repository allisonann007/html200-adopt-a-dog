README file for Adopt-a-dog Pet Adoption website.

Adopt-a-dog is a website for a Seattle-based dog adoption company.  Customers may view dog photos, adoption fees, and make a selection with an adopt button.  A blog is included to give customers ideas for the best way to care and enjoy their canine companions. This webpage was created in order to build a portfolio through the University of Washington's continuing education program for Front-End Software Development.
